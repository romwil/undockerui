OVERVIEW

The Citizen Terminal provides public access to Vault 1848 services and information. This is the standard interface for all Vault residents.

FEATURES

  • Portal Links: Quick access to Vault services (Plex, Radarr, Sonarr, etc.)
  • PIP-BOY NEWS WIRE: Bottom ticker showing Vault news and lore updates
  • GNR (Galaxy News Radio): Wasteland radio streaming (via YouTube live URL)
  • Weather: Current weather for your ZIP code
  • TV/Wall Mode: Large-type layout for living room displays
  • Overseer Access: Classified terminal link (requires hack minigame)

CONTROLS

  • ☰ Menu Button (top-right): Opens settings panel
  • BROADCAST SPEED: Same slider range as Overseer (240–3600 s, default 1800); Citizen saves its own value in this browser.
  • WEATHER ZIP: Enter US ZIP code for weather display
  • GNR PLAY/STOP: Start or stop Galaxy News Radio stream
  • DISK TEMP UNITS: Celsius / Fahrenheit (used on Overseer storage matrix; preference stored here too)
  • TV / WALL MODE: Switch to large-type layout

GNR — GALAXY NEWS RADIO

GNR streams live audio from a configurable YouTube URL. When you click PLAY:

  1. The server runs yt-dlp to resolve the stream URL (may take 10-60 seconds)
  2. The header shows a tuning animation (swirling rings)
  3. Once playing, a live level-bar animation appears in the header
  4. STOP ends playback in this tab and clears the server-side stream cache

Note: Each browser tab has its own audio; stop each tab separately.

PIP-BOY NEWS WIRE

The bottom chyron shows local time, PIP-BOY branding, and rotating Vault news/lore. Speed matches Overseer (☰ menu). News updates from the Vault data feed.

OVERSEER TERMLINK

The "CLASSIFIED: OVERSEER TERMLINK" link in settings triggers a hack minigame. Successfully completing it grants access to the Overseer Command Terminal with full system controls.

TV / WALL MODE

Designed for large displays (/tv): larger type, single-column layout, extra bottom padding for the news chyron, scaled GNR tuning/live widget in the header, and a short on-screen hint pointing to the ☰ menu (lore speed, weather, GNR, disk temp units for Overseer).

OVERVIEW

The Overseer Command Terminal provides full system monitoring and control for Vault 1848. This interface is restricted to authorized personnel only.

STORAGE MATRIX

Displays all array disks, parity drives, and cache drives with real-time status:

  • Temperature: Shown in Celsius. Hot drives (>38°C) highlighted in amber
  • Spin Status: LED indicators show HDD spinning/spun-down or SSD
  • Tooltips: Hover over any drive for detailed information (size, device, filesystem, etc.)

ARRAY & HOST TELEMETRY

Comprehensive system information panel:

  • UPTIME: System uptime timestamp
  • CPU / RAM: Current usage percentages
  • ARRAY: Array state (STARTED, STOPPED, etc.)
  • CAP FREE / USED / TTL: Array capacity in human-readable format
  • CAP % USED: Percentage of array capacity utilized
  • PARITY: Parity check status and progress
  • HOST / UNRAID: Hostname and Unraid version
  • SHARES / DOCKER / VMS: Counts and running status
  • Notifications: Unread alerts and warnings
  • UPS: Battery backup status (if configured)
  • Shares Table: Snapshot of share usage
  • Plugins: Installed plugins sample
  • VMs: Virtual machine status

DOCKER SWARM

Container management and monitoring:

  • Filters: Show ALL or RUNNING ONLY containers
  • Search: Filter by container name, image, or compose stack
  • Stack Filter: Filter by Docker Compose project
  • Container Actions:
    ButtonAction
    UOpen in Unraid Docker UI (external link)
    LOGView last 80 log lines in terminal
    STStart container
    SPStop container
    RSTRestart container
  • Badges: AUTO (auto-start), ORPHAN (orphaned), UPDATE (update available)

TERMINAL / DIAGNOSTIC LOG

The right-side terminal shows:

  • Command history and responses
  • Docker action confirmations
  • GNR radio status
  • Error messages and diagnostics

Scroll to view full history. Terminal is read-only for NOC/TV mode.

GNR — GALAXY NEWS RADIO

Access via ☰ menu:

  • PLAY: Resolves stream with yt-dlp, then starts playback
  • STOP: Stops browser playback in this tab and clears server cache
  • Header signal: Swirling rings while tuning; animated level bars when live
  • DISK TEMP UNITS: Toggle °C / °F for storage matrix (saved in this browser)

PIP-BOY NEWS WIRE

Bottom ticker displays Vault news and lore. Speed adjustable in ☰ menu (BROADCAST SPEED).

NOC / TV MODE

Read-only wall display (/overseer/tv) — widescreen layout:

  • Left column: Storage matrix + full array & host telemetry (scrollable)
  • Right column — Operations deck: integrity ribbon, CPU/RAM/capacity meters, stat tiles (Docker/VM/share counts, parity, flash, plugins), decorative array “scan”, recent notification lines, resource feed echo
  • No per-container list; use FULL CONSOLE → for Docker, logs, GNR, °C/°F, lore speed
  • No terminal/diagnostic log on the wall
  • Narrow screens stack deck below telemetry; chyron stays full width

KEYBOARD SHORTCUTS

None currently. All actions via mouse/touch interface.

TROUBLESHOOTING

  • No disks showing: Check UNRAID_IP and network connectivity
  • Docker actions fail: Verify UNRAID_KEY and GraphQL endpoint
  • GNR won't play: Ensure yt-dlp is installed in container
  • Logs show errors: Check terminal/diagnostic log for details
